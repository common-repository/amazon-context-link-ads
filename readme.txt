=== Amazon Context Links Ads ===
Tags: amazon, links ads
Contributors: Patrick Chia

Add Amazon context link ads to your blog.
Context Links are a quick and convenient way to add links to your page and help you monetize your content. Context Links automatically identify and link relevant phrases within your page content to Amazon products.

== Description ==

Use this plugin to enabled amazon context ads in your blog start earn money from amazon affiliate

== Installation ==

1. Upload to your plugins folder, usually `wp-content/plugins/`
2. Activate the plugin on the plugin screen
4. Add your amazon affiliate ID on option page
